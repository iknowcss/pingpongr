PingPongBoard Spec

Story Cards

  1. As a new player I want to be able to register with my first name, last name, and a user name so that I can participate in games
  2. As a player I want to be able to log in so that I can update my profile and see my stats
  3. As a player I want to be able to update my profile so that I can keep my information up to date
  4. As a player I want to be able to find games that I played in the past so that I can review my progress
  5. As a player I want to be able to see the progression of each game point-by-point so that I can brag to my friends
  6. As a score keeper I want to see all open tables so that I can choose one to score for
  7. As a score keeper I want to register as the only score keeper for a table so that other people don't mess up the score
  8. As a score keeper I want to unregister myself as score keeper from a table so that other people can be score keeper for that table
  9. As a score keeper I want my control over a table to suspend automatically after 15 minutes of inactivity so that the table doesn't stay locked up
  10. As a score keeper I want to create a new ping pong game on my table so that I can track it
  11. As a score keeper I want to be able to update the score of an active game so that I can keep score of the game
  12. As a score keeper I want to be able to "undo" and "redo" points given so that I can track the score more accurately
  13. As a score keeper I want the UI change when a player has won the game so that I can close and save the game results easily
  14. As an audience member I want to see all available tables so that I can see their statuses and access their score boards
  15. As an audience member I want to be able to view the score board for a table so that I may stay up to date with the score and other current game info
  16. As an audience member I want a button to make the score board browser window full screen so that I can view the score board easily
