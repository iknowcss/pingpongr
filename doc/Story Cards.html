PPR-100

As a new player I want to be able to register with my first name, last name, and a user name so that I can participate in games

Acceptance Criteria

  1. On the login page (PPR-), there is a link to a registration page
  2. The registration page has a form where new users may register with the following information:
    1. First and Last name
    2. Nickname
    3. User name
      Note 1: only match [a-zA-Z0-9_-\.]+
    4. Password
    5. Repeat password
    Note 2: The form should have client-side validation to ensure the user's passwords match
  3. After a new user types a user name, the UI will update with the availability of the name
